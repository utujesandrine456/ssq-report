# SSQ Report Generator

Attendance report generator for Sabbath School built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- pnpm

### Installation

First, install dependencies:

```bash
pnpm install
```

### Running the Development Server

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

### Building for Production

To build the application for production:

```bash
pnpm build
```

To start the production server:

```bash
pnpm start
```

## How to Use

1. **Select Report Date**: Choose the date for your attendance report using the date picker
2. **Fill Attendance Data**: Enter attendance numbers for all three families (Ebenezer, Salvation Siblings, Jehovah-Nissi) in the provided fields
3. **Generate Report**: Click "Generate Report" to view the formatted report
4. **Print PDF**: Click "Ctrl + P" to save the report as a PDF file
